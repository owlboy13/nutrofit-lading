Política de Privacidade

Como coletamos, usamos e protegemos seus dados

1. Dados que Coletamos

1.1. Dados de Médicos:

  • Cadastro: Nome, CPF, CRM, especialidade, email, telefone
  • Pagamento: Dados processados por gateway seguro (Stripe/Mercado Pago)
  • Uso: Logs de acesso, configurações, histórico de uso

1.2. Dados de Pacientes:

  • Identificação: Nome, idade, telefone (WhatsApp)
  • Saúde: Dados nutricionais, fotos de refeições, mensagens, áudios
  • Histórico: Evolução nutricional, adesão ao tratamento

1.3. Dados Técnicos:

  • Endereço IP, tipo de navegador, dispositivo
  • Cookies e tecnologias similares
  • Dados de performance e uso da plataforma

2. Como Usamos os Dados

  • Fornecer o serviço: Processar mensagens, gerar relatórios, análise de IA
  • Melhorar a plataforma: Análise de uso, correção de bugs, novos recursos
  • Comunicação: Suporte técnico, atualizações, avisos importantes
  • Segurança: Prevenir fraudes, proteger dados, cumprir obrigações legais
  • Cobrança: Processar pagamentos e emitir notas fiscais

🔐 Importante: Dados de saúde dos pacientes são utilizados APENAS para fornecer o serviço ao médico. Nunca vendemos ou compartilhamos dados de saúde com terceiros.

3. Compartilhamento de Dados

Compartilhamos dados apenas nas seguintes situações:

  • Provedores de Serviço: Hospedagem (AWS/Render), pagamentos (Stripe), email (SendGrid) - sob contratos de confidencialidade
  • APIs de IA: Openai/Anthropic para processamento - dados anonimizados quando possível
  • Obrigações Legais: Quando exigido por lei ou ordem judicial
  • Proteção de Direitos: Para proteger nossos direitos ou de terceiros

❌ Nunca vendemos dados pessoais ou de saúde para terceiros.

4. Armazenamento e Segurança

Medidas de Segurança:

  • Criptografia de dados em trânsito (HTTPS/TLS) e em repouso (AES-256)
  • Backups automáticos diários
  • Autenticação de dois fatores (2FA) disponível
  • Isolamento de dados por médico (multi-tenancy)
  • Logs de auditoria completos
  • Testes de segurança regulares

Retenção de Dados:

  • Dados de pacientes: Mantidos enquanto o médico for cliente ativo
  • Após cancelamento: Dados retidos por 5 anos (conforme CFM) ou até solicitação de exclusão
  • Dados de pagamento: Retidos conforme legislação fiscal (5 anos)

5. Seus Direitos (LGPD)

Você tem direito a:

  • Acesso: Saber quais dados temos sobre você
  • Correção: Atualizar dados incorretos
  • Exclusão: Solicitar remoção dos dados (direito ao esquecimento)
  • Portabilidade: Exportar seus dados em formato estruturado
  • Revogação: Retirar consentimento a qualquer momento
  • Oposição: Opor-se a determinados tratamentos
  • Revisão: Solicitar revisão de decisões automatizadas

Para exercer seus direitos, entre em contato: privacidade@nutrofit.com.br

6. Cookies e Tecnologias Similares

Utilizamos cookies para:

  • Essenciais: Manter sua sessão ativa, autenticação
  • Funcionais: Lembrar preferências e configurações
  • Analytics: Entender como a plataforma é usada (Google Analytics)
  • Marketing: Rastreamento de conversões (desabilitável)

Você pode gerenciar cookies nas configurações do navegador.

⚖️ Conformidade com a LGPD

Lei Geral de Proteção de Dados (Lei nº 13.709/2018)

1. Papéis na Proteção de Dados

NutroFit como Operador:

Para dados de pacientes, a NutroFit atua como operador de dados, processando informações sob instrução do médico (controlador).

🏥 Médico como Controlador:

O médico é o controlador de dados dos pacientes, responsável por:

  • Obter consentimento dos pacientes
  • Definir finalidades do tratamento
  • Responder a solicitações de direitos dos pacientes
  • Garantir conformidade com a LGPD

🏢 NutroFit como Controlador:

Para dados do próprio médico (cadastro, pagamentos), a NutroFit é controlador.

2. Bases Legais para Tratamento

Tratamos dados pessoais com base nas seguintes hipóteses legais:

  • Consentimento: Você concorda com estes termos ao usar a plataforma
  • Execução de Contrato: Necessário para prestar o serviço contratado
  • Legítimo Interesse: Melhorias, segurança e prevenção de fraudes
  • Obrigação Legal: Cumprimento de leis e regulamentos
  • Proteção da Vida: Dados de saúde para tutela da saúde (Art. 7º, VII e Art. 11, II da LGPD)

3. Tratamento de Dados Sensíveis de Saúde

⚠️ ATENÇÃO ESPECIAL:

Dados de saúde são considerados dados pessoais sensíveis pela LGPD e recebem proteção adicional.

Medidas especiais para dados de saúde:

  • Criptografia reforçada (AES-256)
  • Acesso restrito apenas ao médico responsável
  • Logs de auditoria de todos os acessos
  • Anonimização quando possível para análises agregadas
  • Compartilhamento mínimo necessário (apenas com fornecedores de IA sob NDA)
  • Consentimento explícito deve ser obtido pelo médico do paciente

4. Transferência Internacional

Alguns fornecedores estão no exterior (OpenAI, AWS). Garantimos proteção através de:

  • Cláusulas contratuais padrão
  • Certificações ISO 27001, SOC 2
  • Conformidade com LGPD Art. 33
